CC Gen Ultimate

Flutter-based Windows desktop application for generating and translating subtitles using integrated AI models. This project combines modern UI/UX design with advanced machine learning capabilities to streamline subtitle creation workflows.

Main Dashboard & File Processing

The main interface features a clean, intuitive dashboard where users can drag & drop video/audio files. The app automatically detects file formats and displays processing progress with real-time status updates. Background processing allows users to continue working while subtitles are being generated.
