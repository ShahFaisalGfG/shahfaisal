Contacts Manager

A compact Windows desktop utility for importing, normalizing and exporting phone contacts from text, CSV and Excel files. Designed for quick offline processing — detect and group numbers by country code, apply length-based filters, then export or copy lists in several formats.

C# .NET Framework 4.8 WinForms EPPlus (Excel) Inno Setup (installer) CSV / TXT / XLSX
01

Main Window — Load & Parse Contacts

Load contact lists from `.txt`, `.csv` or `.xlsx`. The app scans each line for phone numbers, normalizes formats and shows parsing results with counts and quick actions.

Load TXT / CSV / XLSX Parsing Preview Quick Actions
02

Grouped Contacts View

Contacts are grouped by detected country code into tabs. Click a tab to view all numbers for a country, with counts and quick copy/export options.

Grouped Tabs Per-country Counts Copy / Export
03

Filter & Validation Options

Apply length-based filters, choose to ignore country code when counting digits, and exclude invalid entries. Useful to remove short test numbers and focus on production lists.

Ignore Short / Long Country-code Toggle Validation Rules
05

Export Preview & Save

Preview filtered and sorted numbers before saving. Export options include plain text, importable CSV and Excel (EPPlus). The app preserves formatting suited for common contact importers.

Text / CSV / Excel Preview Before Save Importable Formats
04

Export Options

Choose the desired export format and filename. The exporter supports Excel via EPPlus and generates CSV compatible with major contact importers.

Excel (EPPlus) Importable CSV Save & Open Folder
