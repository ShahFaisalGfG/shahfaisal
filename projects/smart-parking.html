Smart Car Parking System

Final year project: IoT-based smart parking system using ESP32, RFID for entry/exit, and a web dashboard with PHP, MySQL, and JavaScript.


Hardware Integration

ESP32 microcontroller with RFID reader for vehicle detection and servo motors for barriers.

Web Dashboard

Real-time monitoring of parking slots, availability, and entry logs using JavaScript charts.

Database & Logs

MySQL backend storing vehicle entries, timestamps, and slot assignments.
